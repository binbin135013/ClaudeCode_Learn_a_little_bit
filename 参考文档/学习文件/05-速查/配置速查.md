> [!info] 快速查阅
> Claude Code 和 OpenClaw 的关键配置文件路径和格式速查。

---

## Claude Code 配置

### 文件结构

```
~/.claude/                          ← 全局配置目录
├── CLAUDE.md                       ← 全局指令（所有项目）
├── config.json                     ← 全局配置
├── auth-token.json                 ← 认证令牌
├── trusted-directories.json        ← 信任目录列表
├── settings.json                   ← 全局设置
├── cache/                          ← 缓存
└── logs/                           ← 日志

项目根目录/
├── CLAUDE.md                       ← 项目指令（团队共享）
├── .claude/
│   ├── settings.json               ← 项目设置（团队共享）
│   ├── settings.local.json         ← 本地覆盖（不入库）
│   ├── commands/                   ← 自定义命令
│   ├── hooks/                      ← 生命周期钩子
│   └── skills/                     ← 技能包
├── .mcp.json                       ← MCP 服务器配置
└── .claudeignore                   ← 忽略文件列表
```

### settings.json 关键字段

```json
{
  "permissions": {
    "allow": ["Read", "Glob", "Grep", "Bash(npm test *)"],
    "deny": ["Bash(rm -rf /)", "Bash(sudo *)"]
  },
  "sandbox": {
    "filesystem": {
      "allowWrite": ["/tmp", "./src"],
      "allowRead": ["./"],
      "denyRead": [".env"]
    }
  },
  "hooks": {
    "PostToolUse": [{
      "matcher": "Write",
      "hooks": [{ "type": "command", "command": "..." }]
    }]
  }
}
```

### .mcp.json 格式

```json
{
  "mcpServers": {
    "server-name": {
      "command": "npx",
      "args": ["-y", "package-name"],
      "env": { "API_KEY": "${ENV_VAR}" }
    }
  }
}
```

### 环境变量

| 变量 | 说明 |
|------|------|
| `ANTHROPIC_API_KEY` | API 密钥（必需）|
| `ANTHROPIC_BASE_URL` | API 中转站地址 |
| `CLAUDE_MODEL` | 默认模型 |
| `HTTPS_PROXY` | 代理地址 |
| `CLAUDE_AUTO_UPDATE` | 自动更新 (true/false) |

## OpenClaw 配置

### 文件结构

```
~/.openclaw/
├── openclaw.json                   ← 主配置（JSON5 格式）
├── workspace/
│   ├── MEMORY.md                   ← 长期记忆
│   └── memory/                     ← 每日日志
│       └── YYYY-MM-DD.md
└── logs/                           ← 日志
```

### openclaw.json 关键字段

```json5
{
  // 模型配置
  "model": "anthropic/claude-opus-4-6",
  "models": {
    "primary": "anthropic/claude-opus-4-6",
    "fallback": ["openai/gpt-5.4"]
  },

  // Gateway
  "gateway": {
    "port": 18789,
    "token": "your-secret-token"  // 安全认证
  },

  // 消息平台
  "channels": {
    "telegram": { "botToken": "..." },
    "discord": { "botToken": "..." }
  },

  // Agent
  "agents": {
    "default": {
      "systemPrompt": "...",
      "model": "anthropic/claude-sonnet-4-6",
      "skills": ["weather", "github"]
    }
  },

  // 安全
  "security": {
    "allowFrom": ["user-id-1"],
    "dmPolicy": "pairing"  // pairing | allowFrom | open
  }
}
```
