## Claude Code 命令

### 启动与退出

```bash
claude                          # 交互模式
claude "你的指令"                # 单次命令
claude -p "问题"                 # 打印模式（管道友好）
claude --version                # 版本
claude --help                   # 帮助
claude --verbose                # 详细日志
claude --debug                  # 调试模式
claude --new-session            # 新会话
claude --clear-cache            # 清除缓存
/exit                           # 退出（或 Ctrl+C 两次）
```

| 命令 | 功能 | 版本 |
|------|------|------|
| `/help` | 帮助信息 | 初始 |

| `/effort` | 推理深度控制 | v2.1.69+ |

| `/sandbox` | 沙箱隔离 | v2.1.69+ |
| `/fast` | 快速模式切换 | v2.1.52+ |
| `/tp` | 跨设备传送会话 | v2.1+ |
| `/color` | 颜色主题 | v2.1.69+ |
| `/copy` | 复制上一条回复 | v2.1.69+ |
| `/branch` | 创建新分支 | v2.1.69+ |

### 权限相关

```bash
claude                                      # 默认（每次确认）
claude --dangerously-skip-permissions       # ⚠️ 跳过权限
claude --permission-mode plan               # 计划模式（需审批）
claude --permission-mode acceptEdits        # 自动接受编辑
claude --permission-mode bypassPermissions  # ⚠️ 全部跳过
/permissions                                # 查看当前权限
```

### 配置管理

```bash
claude config list                  # 查看所有配置
claude config get <key>             # 查看特定配置
claude config set <key> <value>     # 设置配置
claude install                      # 安装/更新到原生版本
```

## OpenClaw 命令

### 安装与管理

```bash
npm install -g openclaw@latest              # 安装最新版
openclaw onboard                            # 初始化引导
openclaw onboard --install-daemon           # 初始化 + 注册守护进程
openclaw update --channel stable            # 更新（stable/beta/dev）
openclaw doctor                             # 健康检查诊断
```

### 运行与调试

```bash
openclaw gateway --port 18789 --verbose     # 调试模式启动
openclaw gateway --headless                 # 无界面模式
openclaw agent --message "你好"              # 命令行对话
openclaw tui                                # 交互式终端 UI
```

### 消息与配对

```bash
openclaw message send --to +1234 --message "Hi"  # 发送消息
openclaw pairing approve telegram ABC123          # 批准配对
openclaw pairing deny telegram ABC123             # 拒绝配对
```

### 技能管理

```bash
openclaw skill list                         # 列出所有技能
openclaw skill install <name>               # 安装社区技能
openclaw skill create <name>                # 创建新技能
```

## Docker 命令

```bash
docker pull openclaw/openclaw:latest        # 拉取镜像
docker run -d --name openclaw ...           # 运行容器
docker logs -f openclaw                     # 查看日志
docker compose up -d                         # 启动 Compose (v2)
docker compose restart                       # 重启
```
