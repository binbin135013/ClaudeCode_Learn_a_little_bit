# 故障排查手册

> [!summary] 一句话定义
> Claude Code 和 OpenClaw 的**常见问题速查手册**，按错误类型分类，快速定位解决方案。

---

## Claude Code 常见问题

### 安装类

| 问题 | 原因 | 解决 |
|------|------|------|
| `claude: command not found` | PATH 未配置 | 添加 `~/.local/bin` 到 PATH |
| PowerShell 脚本禁止 | 执行策略限制 | `Set-ExecutionPolicy RemoteSigned` |
| Windows Defender 拦截 | 误报 | 添加排除目录 |
| SmartScreen 拦截 | 新签名 | "更多信息" → "仍要运行" |

### 连接类

| 问题 | 原因 | 解决 |
|------|------|------|
| API 连接超时 | 网络问题 | 配置代理或使用中转站 |
| `invalid x-api-key` | Key 无效 | 重新生成 API Key |
| SSL 证书错误 | 证书过期 | 更新系统根证书 |

### 运行类

| 问题 | 原因 | 解决 |
|------|------|------|
| 响应慢 | 上下文过大 | `/clear` + 精简 CLAUDE.md |
| Token 超限 | 单次请求过大 | 分解任务 |
| 中文乱码 | 编码问题 | `chcp 65001` (Windows) |

## OpenClaw 常见问题

### 安装类

| 问题 | 原因 | 解决 |
|------|------|------|
| Node.js 版本低 | 需要 22+ | 升级 Node.js |
| npm 全局安装失败 | 权限问题 | 不用 sudo，用 nvm |
| onboard 失败 | 网络/配置 | `openclaw doctor` 诊断 |

### 运行类

| 问题 | 原因 | 解决 |
|------|------|------|
| Gateway 启动失败 | 端口占用 | 更换端口或杀进程 |
| 消息平台连接失败 | 配置错误 | 检查 Channel 配置 |
| AI 不回复 | API Key 问题 | 检查模型配置 |
| 记忆丢失 | 文件路径错误 | 检查 workspace 路径 |

### Docker 类

| 问题 | 原因 | 解决 |
|------|------|------|
| 容器启动失败 | 配置错误 | `docker logs openclaw` |
| 数据丢失 | 未挂载卷 | `-v ~/.openclaw:/home/node/.openclaw` |
| 外部无法访问 | 防火墙 | 开放 18789 端口 |

## 诊断命令速查

```bash
# Claude Code
claude --version              # 版本检查
claude /doctor                # 系统诊断
claude --verbose              # 详细日志

# OpenClaw
openclaw doctor               # 健康检查
openclaw gateway --verbose    # 调试模式
docker logs -f openclaw       # Docker 日志
```

> [!tip] 排查流程
> 1. 先看错误信息（复制完整报错）
> 2. 匹配上表的已知问题
> 3. 运行诊断命令
> 4. 查看日志文件
> 5. 搜索 GitHub Issues

## 进阶：预防性排查

> [!tip] 预防胜于治疗
> 大部分问题可以通过预防性措施避免。以下是关键预防清单。

### 环境预防
- 运行 `claude --version` 和 `openclaw --version` 定期检查版本
- 使用 `openclaw doctor` 做健康检查 → [[环境搭建指南]]
- 定期更新：`claude update` / `openclaw update --channel stable`

### 配置预防
- CLAUDE.md 控制在 5000 字符以内 → [[CLAUDE-md编写指南]]
- settings.json 变更前先备份 → [[配置速查]]
- 敏感信息用环境变量，不硬编码 → [[安全最佳实践]]

### 常见陷阱速查
- 遇到问题先查 [[常见陷阱与反模式]]，90% 的"bug"其实是配置陷阱
- 排错决策树见 [[常见陷阱与反模式#排错决策树]]
