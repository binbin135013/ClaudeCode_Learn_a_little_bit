这是一个用于学习 Claude Code 源码的 Obsidian 知识库。

```
学习材料/

│   ├── 📊-Dataview-仪表盘.md   # 知识库可视化
│   └── 📋-学习进度追踪.md      # 5周学习计划
```

## 学习路径

推荐按顺序学习：

1. **🚀-快速上手指南** - 先读这个
2. **📐-架构概览** - 理解全局
3. **🔧-Tool系统** - 核心中的核心
4. **🔐-权限系统** - 安全设计
5. **🧠-记忆系统** - 知识库建设
6. **📦-上下文管理** - 上下文优化
7. **⚙️-QueryEngine** - 核心引擎
8. **🪝-Hook系统** - 进阶扩展
9. **📚-Skill系统** - 能力封装
10. **🤝-子Agent与协作** - 多Agent
11. **♻️-核心设计模式** - 设计总结

## Obsidian 插件需求

推荐安装以下插件：

- **Dataview** - 知识库索引和统计
- **Reminder** - 学习打卡提醒
- **Breadcrumbs** - 导航增强
- **Link local** - 更好地处理本地链接

## 学习建议

- 每天学习 1-2 小时
- 按学习进度追踪打卡
- 边学边实践，用 Claude Code 做项目
- 定期复盘，总结所学
