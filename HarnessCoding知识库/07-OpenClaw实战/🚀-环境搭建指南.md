# 🚀 环境搭建指南

> ⏱ 难度: ★☆☆ | 重要性: ★★★ | 推荐学习时间: 1天
> 💡 **新手必看**: 从零开始搭建开发环境！

---

## 前置条件

| 依赖 | 最低版本 | 推荐方式 | 检查命令 |
|------|---------|---------|---------|
| Node.js | 22+ | nvm 安装 | `node --version` |
| npm | 10+ | 随 Node.js | `npm --version` |
| Git | 2.30+ | 官网下载 | `git --version` |
| API Key | - | Anthropic Console | - |

## 一、Claude Code 环境搭建

### 步骤 1：安装 Claude Code

```bash
# 全局安装（推荐）
npm install -g @anthropic-ai/claude-code

# 验证安装
claude --version
```

> [!warning] Windows 用户注意
> 如果 PowerShell 报执行策略错误，先运行：
> `Set-ExecutionPolicy RemoteSigned -Scope CurrentUser`

### 步骤 2：配置 API Key

```bash
# 方式 A：环境变量（推荐，安全）
export ANTHROPIC_API_KEY=sk-ant-xxxxx

# 方式 B：写入 shell 配置，永久生效
echo 'export ANTHROPIC_API_KEY=sk-ant-xxxxx' >> ~/.bashrc
source ~/.bashrc

# Windows PowerShell
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-ant-xxxxx", "User")
```

### 步骤 3：创建全局 CLAUDE.md

```bash
# 创建全局配置目录
mkdir -p ~/.claude

# 创建全局 CLAUDE.md
cat > ~/.claude/CLAUDE.md << 'EOF'
# 个人偏好
- 使用中文交流
- 代码注释用中文
- 优先使用 TypeScript

# 常用技术栈
- 前端：React + Next.js
- 后端：Node.js + Express
- 数据库：PostgreSQL + Prisma
EOF
```

### 步骤 4：安装推荐 MCP Servers

常用 MCP Server 速查：

| Server | 用途 | 安装 |
|--------|------|------|
| filesystem | 文件读写 | `npx @anthropic-ai/mcp-filesystem` |
| github | PR/Issue 管理 | `npx @anthropic-ai/mcp-github` |
| postgres | 数据库查询 | `npx @anthropic-ai/mcp-postgres` |

### 步骤 5：测试 Claude Code

```bash
# 创建测试项目
mkdir ~/test-cc && cd ~/test-cc
git init

# 启动 Claude Code
claude

# 输入测试指令
> 你好，请帮我创建一个 hello.txt 文件，内容是 "Hello Claude Code"

# 确认文件已创建
ls hello.txt
```

## 二、OpenClaw 环境搭建

### 步骤 1：安装 OpenClaw

```bash
# 全局安装
npm install -g openclaw

# 验证安装
openclaw --version
```

### 步骤 2：运行 onboard 向导

```bash
# 交互式初始化配置
openclaw onboard
```

向导会引导你完成：
- 选择 AI 模型提供商
- 输入 API Key
- 配置基本信息（语言、时区等）

> [!warning] 常见问题
> 如果 onboard 失败，运行 `openclaw doctor` 查看诊断信息。
> 大多数失败原因是 Node.js 版本过低（需要 22+）。

### 步骤 3：测试 OpenClaw

```bash
# CLI 模式测试
openclaw agent --message "你好，请自我介绍"

# 预期输出：AI 回复自我介绍
```

## 三、CC + OC 集成搭建

### 步骤 1：共享 API Key

```bash
# 确保环境变量已设置（Claude Code 步骤 2 中已配置）
echo $ANTHROPIC_API_KEY

# OpenClaw 的 openclaw.json 中已通过 ${ANTHROPIC_API_KEY} 引用
# 两者自动共享同一个 Key
```

### 步骤 2：配置 MCP 桥接

```bash
# 安装 mcporter（MCP 桥接工具）
npm install -g mcporter
```

> [!warning] 前提条件
> OpenClaw Gateway 必须处于运行状态（`openclaw gateway`）才能桥接成功。

## 环境验证清单

- [ ] `claude --version` 输出正确版本号

- [ ] `mcporter` 已安装
