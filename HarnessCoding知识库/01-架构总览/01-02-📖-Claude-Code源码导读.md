## 如何使用参考代码进行源码学习

### 参考代码目录结构

```
cc-learn/
├── 参考代码/
│   ├── claude-code/                    # Claude Code 官方源码
│   └── Claude-Code-x-OpenClaw-Guide/   # 对比参考实现
```

### 学习方法

#### 1. 建立全局视图

首先浏览目录结构，理解代码组织方式：

```bash
# 查看源码目录树
tree -L 3 参考代码/claude-code/src
```

#### 2. 从核心文件开始

按照本文档的优先级顺序阅读，从 `query.ts` 开始建立全局理解。

#### 3. 交叉验证

当你遇到一个设计决策时，可以查看 `Claude-Code-x-OpenClaw-Guide` 中的对比实现，帮助理解为什么 Claude Code 这样设计。

#### 4. 结合知识库

每个文件都标注了对应的知识库章节。在阅读源码前，先浏览对应的知识库文档，可以事半功倍。

#### 5. 实践验证

尝试修改代码并运行测试，理论结合实践才能真正理解。

### 源码阅读辅助工具

- **grep / ripgrep**: 在源码中搜索特定函数或模式
- **AST 工具**: 理解代码结构关系
- **调试器**: 设置断点跟踪执行流程
- **知识库**: 利用已有的架构文档加速理解
