## 如何使用参考代码学习

### 学习路径建议

1. **入门**：`learn-coding-agent` → `cc-src-learning`
2. **核心**：`claude-code` 官方源码 + `claw-code-main`
3. **进阶**：`oh-my-claudecode` 编排框架 + `everything-claude-code`
4. **深化**：`ai-agent-deep-dive` PDF 报告

### 阅读技巧

| 场景 | 推荐仓库 | 阅读方式 |
|------|----------|----------|
| 理解核心架构 | `claude-code` | 从入口文件开始，逐步深入 |
| 实践项目开发 | `claw-code-main` | 对照实战章节边做边学 |
| 查找设计模式 | `everything-claude-code` | 按需检索，案例驱动 |
| 理解多 Agent | `oh-my-claudecode` | 画架构图，理清关系 |

### 源码阅读工具

- **推荐 IDE**：VS Code + Claude Extension
- **代码搜索**：`claude-code-analysis` 中的索引文件
- **架构可视化**：参考 `everything-claude-code` 中的图表
