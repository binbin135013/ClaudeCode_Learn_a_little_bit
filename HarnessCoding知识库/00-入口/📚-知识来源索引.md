## 贡献来源

### 如何贡献新来源

如果你发现了有价值的学习资源，欢迎贡献到本索引。

### 贡献步骤

1. **提交建议**：在知识库的问题跟踪中提出新来源建议
2. **包含信息**：
   - 来源名称
   - 来源类型（GitHub / 官方文档 / 文章 / 视频等）
   - 简短描述
   - 原始链接
3. **审核后收录**：维护者审核后会将来源添加到相应分类

### 收录标准

- ✅ 与 Claude Code / OpenClaw / Harness Coding 直接相关
- ✅ 内容准确、来源可靠
- ✅ 非商业广告或推广内容
