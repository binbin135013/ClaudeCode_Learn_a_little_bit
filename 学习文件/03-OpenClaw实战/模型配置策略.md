# 模型配置策略

> [!summary] 一句话定义
> OpenClaw 支持 **29+ AI 模型提供商**，支持模型切换、故障转移、多模型并存，灵活应对不同场景。

---

## 推荐配置

| 场景 | 推荐模型 | 理由 |
|------|---------|------|
| **日常使用** | Claude Haiku 4.5 / GPT-5.2-mini | 速度快，成本低，适合日常对话 |
| **成本优先** | Claude Sonnet 4.6 | 性价比高 |
| **隐私优先** | Ollama 本地模型 | 零外部调用 |
| **中国用户** | 通义千问 / DeepSeek | 中文优化 |

## 故障转移配置

```json5
// openclaw.json
{
  "agents": {
    "defaults": {
      "model": {
        "primary": "anthropic/claude-opus-4-6",
        "fallbacks": [
          "openai/gpt-5.4",
          "ollama/llama3.1"
        ]
      }
    }
  }
}
```

> [!tip] 配置策略
> 1. **主模型** — 选最强的，处理大部分任务
> 2. **备用模型** — 选便宜的，主模型挂了自动切
> 3. **本地模型** — 隐私敏感场景使用
> 4. **中国模型** — 网络受限时使用
