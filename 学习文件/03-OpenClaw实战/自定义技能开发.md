# 自定义技能开发

> [!summary] 一句话定义
> OpenClaw 技能是**Markdown 指令文件**，教 AI 如何组合使用工具完成复杂任务。

---

## 技能三层级

| 层级 | 来源 | 说明 |
|------|------|------|
| Bundled | 核心仓库 | 50+ 内置，开箱即用 |
| Managed | ClawHub | 社区技能市场 |
| Workspace | 本地 | 自己开发 |

## 技能本质

```
工具 = 单个原子操作（读文件、发请求）
技能 = 组合工具完成复杂任务的知识
```

**示例**：GitHub Issue 管理 = 搜索 Issue + 读取详情 + 添加评论 + 修改标签

## 热门内置技能

| 类别 | 技能 | 功能 |
|------|------|------|
| 办公 | `gog` | Google Workspace |
| | `notion` | Notion 管理 |
| | `obsidian` | Obsidian 笔记 |
| 开发 | `coding-agent` | 编程助手 |
| | `github` | GitHub 管理 |
| 多媒体 | `spotify-player` | 音乐控制 |
| | `peekaboo` | 截图 |
| 实用 | `weather` | 天气查询 |
| | `skill-creator` | 创建技能的技能！|

## 开发自定义技能

### 基本结构

```markdown
# 我的自定义技能

## 概述
描述这个技能做什么...

## 工具要求
- tool_1: 用于xxx
- tool_2: 用于xxx

## 执行流程
1. 步骤一...
2. 步骤二...
3. 步骤三...

## 输出格式
...
```

### 放置位置

```
~/.openclaw/workspace/skills/
└── my-custom-skill.md
```

> [!tip] 开发建议
> 1. 先用 `skill-creator` 技能辅助创建
> 2. 参考内置技能的写法（`skills/` 目录）
> 3. 测试通过后可发布到 ClawHub

## ClawHub 发布

ClawHub (clawhub.com) 是技能和插件市场：
- 社区开发者发布技能
- 其他用户一键安装
- 官方鼓励新技能优先发布到 ClawHub
